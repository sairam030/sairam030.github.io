## Running Locally

```bash
# Install dependencies
npm install
# or
yarn

# Start development server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Building for Production

```bash
npm run build
# or
yarn build
```
